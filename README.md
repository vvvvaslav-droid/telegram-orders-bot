# Telegram-бот для сбора заявок (с базой данных и админ-панелью)

## 📌 Что делает
Бот принимает заявки от пользователей (имя и телефон), сохраняет их в базу данных SQLite и позволяет заказчику просматривать и очищать список через админ-панель (по паролю).

## 🛠️ Как запустить
1. Установите Python 3.12+
2. Установите библиотеку:  
   `pip install python-telegram-bot==20.8`
3. Получите токен у [@BotFather](https://t.me/BotFather) и вставьте его в код.
4. Запустите:  
   `python bot.py`
5. В Telegram напишите `/start` и следуйте инструкциям.

## 🔐 Админ-панель
- Введите пароль `7428` (замените в коде на свой) вместо имени, чтобы открыть админ-панель.
- Доступны кнопки:
  - **«Заявки»** — просмотр всех заявок
  - **«Очистить базу»** — удаление всех заявок
